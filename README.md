# Network Science Assignments

Coursework repository for network science assignments covering graph statistics,
random graph models, signed networks, centrality, community structure, and
network visualization.

## Repository Layout

- `Assignment1/` contains graph-analysis notebooks, reports, datasets, and
  generated plots for degree distributions, clustering coefficients,
  path-length distributions, and visualizations.
- `Assignment2/` contains additional graph-analysis notebooks, reports, and
  datasets including Facebook and biological network data.
- `Assignment3/` contains directed and signed network analysis, configuration
  model and edge-swapping experiments, degree-correlation comparisons, and
  political network data.

## Requirements

Use Python with Jupyter Notebook support. Typical dependencies for these
notebooks include NetworkX, NumPy, pandas, matplotlib, and related scientific
Python packages.

## Usage

Open the notebook for the question you want to inspect, then read the matching
PDF report for methodology and results. Datasets and generated plots are stored
beside the notebook that uses them.

## Notes

The assignment folders are self-contained. Large datasets are stored in the
question directory where they are used.
